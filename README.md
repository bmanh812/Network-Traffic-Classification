# 📊 Phân Loại Traffic Mạng (Network Traffic Classification)
Dự án cuối khóa môn Thực tập cơ sở - Nhóm: Mạnh, Hùng, Hiếu.

## 🌟 Điểm nổi bật
- **Mô hình:** CNN 1D (Deep Learning).
- **Kỹ thuật đặc biệt:** Sử dụng **SMOTE thông minh** với ngưỡng cân bằng 2000 mẫu để chống Overfitting.
- **Kết quả:** Đưa `Accuracy` và `Val_Accuracy` về mức sát nhau (~63-65%), đảm bảo mô hình không "học vẹt".

## 🛠️ Công nghệ sử dụng
- Python, TensorFlow/Keras, Scikit-learn, Imbalanced-learn.
